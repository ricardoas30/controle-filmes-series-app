Este é um projeto final da disciplina de frameworks web na linguagem em Angular, do curso de especialização em linguagem Java da UTFPR - Universidade Téc. Fed. do Paraná. Esta aplicação tem como objetivo de efetuar o cadastro, alteração, deleção e informar se foram assistidos. O usuário terá seus filmes e séries listados e poderá controlar os que foram assistidos. O projeto encontra-se atualmente em fase de desenvolvimento e poderá ser acompanhado pelo repositório github já compartilhado.
